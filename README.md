Anime Search App

A simple React + TypeScript app that allows users to search for anime and view details using the Jikan API.

Live Demo

View the hosted app here -> https://hilarious-starship-7e7f8c.netlify.app/

Features

- Search anime by title with instant search (debounced)
- View detailed information for each anime
- Pagination for browsing results
- Loading and empty states for better UX
- Responsive layout for desktop and mobile
- Built with React, Redux, and TypeScript

Tech Stack

- React 18
- TypeScript
- Redux Toolkit
- React Router DOM
- Tailwind CSS
- Vite

How to Run Locally
npm install
npm run dev


Then open http://localhost:4000
